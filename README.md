
# Scapy XSS Lab Assistant

Репозиторий содержит материалы для выполнения домашнего задания по курсу Python (тема "Взаимодействие в WWW").

## Состав проекта
- `scapy_sniff.py` - анализатор трафика.
- `xss_injector.py` - инструмент для проведения атак.
- `REPORT.md` - итоговый отчет.

## Запуск
Для работы скриптов требуется Scapy и права суперпользователя:
```bash
sudo python3 scapy_sniff.py
```
